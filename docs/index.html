What is an Equipment Suffix Code?

Controllers need to know certain things about your aircraft quickly that affect the way your flight is handled.
Thankfully we have a system that tells ATC all of that information with a simple code generally referred to as a Suffix Code.
In the following example, the "/L" is the equipment suffix code:
B738/L = A Boeing 737-800 aircraft that is RVSM, GNSS capable and it has a Mode C transponder.
